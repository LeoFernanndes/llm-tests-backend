## Dependencies
1. PgVector 16
2. Python 3.11

## Setup
1. Create a .env file following .env.sample example.  
2. Install dependencies from requirements.txt.  
3. Run `fastapi run src/main.py --port 8000` to spin up the server on localhost:8000.
